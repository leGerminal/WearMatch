# WearMatch AI — iPhone PWA

## Objectif
Version ultra-simple pour iPhone :
- ouvrir dans Safari ;
- ajouter à l’écran d’accueil ;
- utiliser caméra / galerie ;
- avatar ;
- bibliothèque ;
- WearScore ;
- WearVision en mode démo prêt pour IA réelle ;
- battles.

## Installation sur iPhone
1. Mettre ce dossier en ligne sur Vercel, Netlify ou GitHub Pages.
2. Ouvrir l’URL sur Safari iPhone.
3. Bouton Partager.
4. “Sur l’écran d’accueil”.
5. Ajouter.

## Important
Cette version ne dépend pas de Node.js pour se lancer.
La génération IA réelle sera branchée ensuite sur le backend Cloud.
